# PayolaSpy

## Install

In your `Gemfile`

```ruby
gem 'payola_spy'
```

## Mount the engine

In `config/routes.rb`

```ruby
mount PayolaSpy::Engine, at: "/payola_spy"
```

## Watch the action

Now you can keep track of Payola activity by visiting your app at: `/payola_spy`
